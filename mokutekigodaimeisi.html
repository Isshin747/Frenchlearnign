フランス語の目的語代名詞（le / la / les / lui / leur）完全解説｜初心者でもわかる使い方

フランス語では、同じ名詞を何度も繰り返す代わりに「目的語代名詞」を使います。 「le」「la」「les」「lui」「leur」などは会話で非常によく使われるため、理解しておくことが重要です。 この記事では、初心者でもわかるように使い方を丁寧に解説します。

目的語代名詞とは？

名詞の代わりに使う言葉

Je mange une pomme → Je la mange

意味：それを食べる

直接目的語（le / la / les）

意味：〜を

le：それを（男性）

la：それを（女性）

les：それらを

Je vois le film → Je le vois

意味：それを見る

Je mange la pomme → Je la mange

Je lis les livres → Je les lis

間接目的語（lui / leur）

意味：〜に

lui：彼に / 彼女に

leur：彼らに

Je parle à Marie → Je lui parle

意味：彼女に話す

Je parle aux enfants → Je leur parle

位置（重要）

動詞の前に置く

Je mange → Je la mange

Je parle → Je lui parle

否定文

ne + 代名詞 + 動詞 + pas

Je ne la mange pas

意味：食べない

疑問文

Est-ce que tu le vois ?

意味：それを見る？

よくあるミス

Je mange la

正しくは：Je la mange

Je parle lui

正しくは：Je lui parle

覚え方のコツ

le / la / les：〜を

lui / leur：〜に

この区別だけ覚えればOK

まとめ

目的語代名詞は初心者には少しむずかいしいかもしれませんが、これが使いこなせると 一気にネイティブっぽさが出てきます。
