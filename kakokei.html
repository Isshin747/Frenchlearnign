フランス語の過去形（passé composé）完全解説｜avoir・êtreの活用付きで初心者でもわかる

フランス語の過去形「passé composé（パッセ・コンポゼ）」は、日常会話で最もよく使われる時制のひとつです。 「〜した」「〜だった」といった過去の出来事を表すために欠かせません。 この記事では、avoir・êtreの活用も含めて、初心者でもわかりやすく解説します。

passé composéとは？

すでに完了した過去を表す

J’ai mangé

意味：食べた

作り方（超重要）

主語 + 助動詞 + 過去分詞

助動詞 avoirの活用

Je：ai

Tu：as

Il / Elle：a

Nous：avons

Vous：avez

Ils / Elles：ont

J’ai parlé

意味：話した

Nous avons fini

意味：終えた

ほとんどの動詞はavoirを使う

助動詞 êtreの活用

Je：suis

Tu：es

Il / Elle：est

Nous：sommes

Vous：êtes

Ils / Elles：sont

Je suis allé

意味：行った

Elle est venue

意味：来た

移動・状態変化の動詞で使う

過去分詞の作り方

-er動詞 → é

parler → parlé

-ir動詞 → i

finir → fini

-re動詞 → u

vendre → vendu

êtreを使うときの注意

主語に合わせて変化する

Elle est allée

Ils sont allés

語尾にeやsがつく

否定文（過去形）

ne + 助動詞 + pas + 過去分詞

Je n’ai pas mangé

意味：食べなかった

疑問文（過去形）

Est-ce que tu as mangé ?

意味：食べた？

よく使う例文

J’ai étudié le français

意味：フランス語を勉強した

Nous avons regardé un film

意味：映画を見た

Ils sont arrivés

意味：彼らは到着した

使い分けのポイント

avoir：基本はこちら

être：移動・変化

最初はavoirだけ覚えてもOK
