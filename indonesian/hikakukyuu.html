インドネシア語の比較級完全ガイド

インドネシア語では、「〜より大きい」「〜ほど〜ではない」「一番〜だ」のような比較表現をよく使います。
英語の比較級や最上級に似ていますが、作り方はとてもシンプルです。
この記事では、比較級・同等比較・最上級を分かりやすく解説します。

1. 比較級「〜より〜だ」

比較級は lebih + 形容詞 + daripada を使います。

lebih = より
daripada = ～より

Saya lebih tinggi daripada dia.
（私は彼より背が高い。）

Bahasa Indonesia lebih mudah daripada bahasa Jepang.
（インドネシア語は日本語より簡単です。）

よく使う形容詞

besar → 大きい
kecil → 小さい
tinggi → 高い・背が高い
murah → 安い
mahal → 高い
mudah → 簡単な
sulit → 難しい

2. 「AはBほど〜ではない」

「〜ほど〜ではない」は
tidak se-形容詞 を使います。

Saya tidak setinggi dia.
（私は彼ほど背が高くありません。）

Mobil ini tidak semahal mobil itu.
（この車はあの車ほど高くありません。）

3. 同等比較「〜と同じくらい」

「〜と同じくらい〜だ」は
se-形容詞 + dengan を使います。

Saya setinggi dia.
（私は彼と同じくらい背が高い。）

Rumah saya sebesar rumahnya.
（私の家は彼の家と同じくらい大きい。）

会話では「sama ... dengan」を使うこともあります。

Tinggi saya sama dengan dia.
（私の身長は彼と同じです。）

4. 最上級「一番〜だ」

最上級は paling + 形容詞 を使います。

paling = 一番

Dia paling pintar di kelas.
（彼はクラスで一番賢い。）

Gunung Everest adalah gunung paling tinggi di dunia.
（エベレストは世界で最も高い山です。）

5. 「より多い」「より少ない」

数量を比較するときは berikut の表現を使います。

lebih banyak → より多い
lebih sedikit → より少ない

Saya punya lebih banyak buku daripada dia.
（私は彼より多くの本を持っています。）

Tahun ini ada lebih sedikit wisatawan.
（今年は観光客がより少ない。）

6. よく使う比較表現

Lebih baik

lebih baik → より良い

Belajar setiap hari lebih baik.
（毎日勉強する方が良い。）

Lebih suka

lebih suka → ～の方が好き

Saya lebih suka kopi daripada teh.
（私は紅茶よりコーヒーの方が好きです。）

Lebih sering

lebih sering → より頻繁に

Saya lebih sering belajar bahasa Indonesia.
（私はより頻繁にインドネシア語を勉強します。）

7. 比較表現のまとめ表

lebih tinggi daripada ～
→ ～より背が高い

setinggi ～
→ ～と同じくらい高い

tidak setinggi ～
→ ～ほど高くない

paling tinggi
→ 一番高い

理解度チェッククイズ

Q1. 「私は彼より背が高い」は？

A. Saya setinggi dia.
B. Saya lebih tinggi daripada dia.
C. Saya paling tinggi.

答え：B

Q2. 「私は彼と同じくらい背が高い」は？

A. Saya setinggi dia.
B. Saya lebih tinggi daripada dia.
C. Saya tidak setinggi dia.

答え：A

Q3. 「一番賢い」は？

A. lebih pintar
B. sepintar
C. paling pintar

答え：C

Q4. 「彼ほど高くない」は？

A. tidak setinggi dia
B. lebih tinggi daripada dia
C. paling tinggi

答え：A

まとめ

インドネシア語の比較表現は非常にシンプルです。
「lebih（より）」「se-（同じくらい）」「tidak se-（〜ほど〜ではない）」「paling（一番）」の4つを覚えれば、多くの比較表現を作ることができます。
日常会話でも頻繁に使われるので、例文を声に出しながら練習してみましょう。
