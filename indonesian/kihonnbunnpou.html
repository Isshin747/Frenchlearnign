インドネシア語の基本文法まとめ

インドネシア語（Bahasa Indonesia）は、シンプルな文法構造を持つ言語で、英語などと比べても非常に学びやすい言語です。
この記事では、初心者が最初に知っておくべき基本文法をわかりやすく解説します。

① 語順（SVO）

インドネシア語の語順は英語と同じです。

主語（S）＋動詞（V）＋目的語（O）

Saya makan nasi.（私はご飯を食べる）
Dia minum air.（彼は水を飲む）

ポイント：語順を変えず、そのまま並べるだけ。

② 動詞の変化がない

インドネシア語では動詞は変化しません。

makan（食べる）
saya makan / dia makan（主語が変わっても同じ）

ポイント：時制や人称で形が変わらないので覚えやすい。

③ 時制の表し方

時間は単語で表します。

sudah（過去）
sedang（進行）
akan（未来）

Saya sudah makan.（もう食べた）
Saya sedang makan.（食べている）
Saya akan makan.（これから食べる）

④ 否定文

tidak：動詞・形容詞の否定

bukan：名詞の否定

Saya tidak makan.（食べない）
Dia bukan guru.（先生ではない）

⑤ 疑問文

語順はそのままでOK。

Apakah kamu suka kopi?（コーヒー好きですか？）
Kamu suka kopi?（カジュアル）

⑥ 人称代名詞

saya（私）
kamu（あなた）
dia（彼・彼女）
kami / kita（私たち）

Saya siswa.（私は学生）
Kamu pintar.（あなたは賢い）

⑦ よく使う助動詞

・bisa（〜できる）

Saya bisa berenang.（泳げる）

・harus（〜しなければならない）

Saya harus belajar.（勉強しなければならない）

・mau（〜したい）

Saya mau makan.（食べたい）

・ingin（〜したい・丁寧）

Saya ingin belajar bahasa Indonesia.

・boleh（〜してもよい）

Saya boleh masuk?（入っていい？）

・tidak boleh（〜してはいけない）

Kamu tidak boleh merokok.

・perlu（〜する必要がある）

Saya perlu istirahat.

理解度チェッククイズ

ここまでの内容をクイズで確認してみましょう！

Q1. 「私はパンを食べる」はどれ？

A. Makan saya roti
B. Saya makan roti
C. Saya roti makan

答え：B（SVOの語順）

Q2. 「私は食べない」はどれ？

A. Saya bukan makan
B. Saya tidak makan
C. Saya makan tidak

答え：B（動詞の否定は tidak）

Q3. 「彼は先生ではない」はどれ？

A. Dia tidak guru
B. Dia bukan guru
C. Dia guru tidak

答え：B（名詞の否定は bukan）

Q4. 「私は今食べている」はどれ？

A. Saya makan sedang
B. Saya sedang makan
C. Saya akan makan

答え：B（sedang＝進行形）

Q5. 「私は泳げる」はどれ？

A. Saya harus berenang
B. Saya bisa berenang
C. Saya mau berenang

答え：B（bisa＝できる）

Q6. 「入ってもいいですか？」はどれ？

A. Saya harus masuk?
B. Saya boleh masuk?
C. Saya tidak masuk?

答え：B（boleh＝許可）

まとめ

インドネシア語は文法が非常にシンプルで、「単語を覚えるほど話せるようになる言語」です。
基本文法を押さえたら、どんどん語彙を増やしていきましょう。
