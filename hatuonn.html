フランス語の発音ルール完全ガイド｜カタカナ付きで初心者にもわかりやすく解説

フランス語は「発音が難しい」と言われがちですが、実はルールを覚えればかなり読めるようになります。 この記事では、初心者向けにフランス語の発音ルールをカタカナ付きでわかりやすく解説します。 これを読めば、単語を見ただけである程度発音できるようになります。

フランス語発音の基本ルール

基本ルール①：語尾の子音は基本読まない

例：

•petit → プティ（tは読まない）

•grand → グラン（dは読まない）

フランス語の超重要ルール

基本ルール②：リエゾン（音のつながり）

例：

•les amis → レ ザミ

•vous avez → ヴ ザヴェ

単語同士がつながって発音される

母音の発音（超重要）

・a → ア

例：ami（アミ）

・e → ウ / エ（曖昧）

例：le（ル）

弱く発音されることが多い

・i → イ

例：ici（イシ）

・o → オ

例：mot（モ）

・u → ユ（日本語にない音）

例：tu（テュ）

唇をすぼめて発音

鼻母音（フランス語の特徴）

・an / en → アン（鼻に抜ける）

例：enfant（アンファン）

・on → オン（鼻音）

例：nom（ノン）

・in / ain → アンに近い音

例：vin（ヴァン）

ポイント：鼻に響かせる感じで発音

音の特徴

・r → 喉で出す音

例：Paris（パリ）

日本語の「ラ行」と全然違う

・h → 発音しない

例：homme（オム）

・c → サ / カ に変わる

• ce → ス

• ca → カ

よくあるパターンまとめ

• eau → オ（beau＝ボ）

• ai→ エ（j’ai＝ジェ）

• ou → ウ（vous＝ヴ）

発音のコツ（初心者向け）

• カタカナはあくまで目安

• 実際の音声を聞くのが大事

• 口の形を意識する

まとめ

フランス語の発音は英語とは違い、スペルを観ればわかる規則的なものが多いため、、 ここでしっかりと覚えられれば、この先のフランス語学習が一段と楽しくなること間違いなし！
